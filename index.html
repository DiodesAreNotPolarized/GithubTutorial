Updating the Repository

After you've made changes on your computer, you're going to want to also change the files in the online reposity. To do this, there are a few commands you will need to use.
First, you will have to git add -A. This command adds all the changes you've changed to be commited later.
Then, you'll need to commit to your repo with git commit -m "commit message". You should specify what you have changed in this commit.
